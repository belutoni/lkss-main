Contributing
------------

Since this is an open-source project, all kinds of contributions are encouraged
and highly appreciated. Your contributions will help improve the infrastructure
and guides for future editions of the school.

If you encounter a bug or issue, please make sure that you file a report under
the `Issues`_ tab of the `main`_ project repository. Additionally, we encourage
students to try to fix the issues themselves by submitting a pull request.

.. _Issues: https://github.com/NXP-Research/lkss-main/issues
